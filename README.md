# Sequelizer

### A desktop application to export sequelize models automatically and visualy.


```bash
# Clone this repository
git clone https://github.com/andyforever/sequelizer.git
# Go into the repository
cd sequelizer
# Install dependencies
npm install
# Run the app
npm start
```
